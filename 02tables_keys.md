# Tables and Keys

## Tables
column --> attribute
row --> record

## Keys
### primary key
- the unique key for rows in this table
### Composite key (Compound key)
- when multiple columns to uniquely identify one row, these columns make up a composite key
### surrogate key
only has meaning in this table
### natural key
has meaning outside the table, may be used in other places
### foreign key
is primary key of another table, linking this row to another row (in another table or in the same table)
